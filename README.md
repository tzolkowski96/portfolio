# Portfolio - Tobin Zolkowski

Personal portfolio website for Tobin Zolkowski, a Data Analyst specializing in data engineering and ETL.

## Features
- Single-page responsive design
- Dark theme with high accessibility
- Interactive canvas background (particle system)
- Performance optimized (conditional animations, light weight)
- Accessible color palette and focus states

## Tech Stack
- HTML5
- CSS3 (Custom properties, Flexbox/Grid)
- Vanilla JavaScript
- Google Fonts (Newsreader, Space Mono, Instrument Serif)

## Deployment
This site is hosted on GitHub Pages. To enable it:
1. Go to your repository settings on GitHub.
2. Navigate to the **Pages** section in the sidebar.
3. Under **Build and deployment**, select `Deploy from a branch`.
4. Choose the `main` branch and the `/ (root)` folder.
5. Click **Save**.

Any changes pushed to the `main` branch will be automatically deployed.
